---

## 介绍

HTML 无障碍访问（HTML Accessibility）是一门关于创建可访问的 HTML 文档的学科。无障碍访问是指能够让残障人士（包括那些不能使用键盘、屏幕阅读器或其他辅助技术的人）能够顺利地使用 Web 内容。无障碍访问的目标是确保 Web 内容能够被广泛阅读、理解和操作。

## 无障碍访问的原则

无障碍访问的原则是：

1. 易用性：Web 内容应该易于使用，包括键盘导航、屏幕阅读器支持、颜色对比度、字体大小、字体选择、链接样式等。
2. 易理解性：Web 内容应该易于理解，包括用词清晰、标注清楚、图像描述清楚、文本对齐、语音朗读等。
3. 易访问性：Web 内容应该易于访问，包括提供足够的文字大小、颜色对比度、字体选择、链接样式、图像描述、键盘快捷键等。
4. 无障碍阅读：Web 内容应该能够被广泛阅读，包括使用语音朗读、缩放、字体大小调整、颜色对比度调整等。
5. 无障碍操作：Web 内容应该能够被无障碍操作，包括键盘导航、屏幕阅读器支持、语音命令、颜色对比度调整、字体大小调整等。

## 无障碍访问的工具

无障碍访问的工具有：

1. 键盘导航：键盘导航是指通过键盘上的方向键、Tab 键、Enter 键等，来控制页面的浏览。
2. 屏幕阅读器：屏幕阅读器是一种辅助技术，它能够让残障人士通过声音来阅读页面内容。
3. 颜色对比度：颜色对比度是指页面上文本、背景、按钮等元素的颜色之间的对比度。
4. 字体大小：字体大小是指页面上文本的大小。
5. 字体选择：字体选择是指页面上文本的字体。
6. 链接样式：链接样式是指页面上链接的颜色、样式、下划线等。
7. 图像描述：图像描述是指页面上图像的文字说明。
8. 键盘快捷键：键盘快捷键是指页面上常用的快捷键。
9. 语音命令：语音命令是指通过语音控制页面的浏览。
10. 缩放：缩放是指页面的大小调整。
11. 颜色对比度调整：颜色对比度调整是指页面上元素的颜色对比度的调整。
12. 字体大小调整：字体大小调整是指页面上文本的大小的调整。

## 无障碍访问的注意事项

1. 语义化：HTML 代码应该语义化，使用正确的标签来描述内容。
2. 颜色对比度：颜色对比度应该足够，避免使用过多的颜色。
3. 字体大小：字体大小应该足够，避免使用过大的字体。
4. 链接样式：链接样式应该清晰，避免使用太多的颜色。
5. 图像描述：图像描述应该清晰，避免使用图片代替文字。
6. 键盘快捷键：键盘快捷键应该一致，避免使用不一致的快捷键。
7. 语音命令：语音命令应该一致，避免使用不一致的命令。
8. 缩放：缩放应该足够，避免使用过大的字体。
9. 颜色对比度调整：颜色对比度调整应该足够，避免使用过多的颜色。
10. 字体大小调整：字体大小调整应该足够，避免使用过大的字体。
11. 键盘导航：键盘导航应该一致，避免使用不一致的键盘导航。
12. 屏幕阅读器：屏幕阅读器应该能够正常运行，避免使用不支持的屏幕阅读器。
