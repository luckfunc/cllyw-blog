HTML（HyperText Markup Language）是构建网页的基础语言。本章将系统学习 HTML 的核心概念和使用技巧。

## 学习目标

- 掌握 HTML 的基本结构和语法
- 理解语义化标签的重要性
- 学会创建表单和表格
- 掌握无障碍访问的基本原则
- 能够编写符合标准的 HTML 代码

## HTML 版本历史

- HTML 1.0 (1991)
- HTML 4.01 (1999)
- XHTML 1.0 (2000)
- HTML5 (2014) - 当前主流标准
- HTML5.1, HTML5.2, HTML5.3 (后续版本)

## 基础结构

HTML 文档由以下部分组成：

- 文档类型声明（DOCTYPE）
- 元素（Elements）
- 属性（Attributes）
- 内容（Content）

## 学习建议

1. **理解而非记忆**：理解标签的语义，而不是死记硬背
2. **动手实践**：每个知识点都要实际编写代码
3. **关注标准**：遵循 W3C 标准编写代码
4. **渐进增强**：从基础结构开始，逐步添加功能
