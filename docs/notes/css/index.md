# CSS 学习

CSS（Cascading Style Sheets）是网页的样式描述语言，用于控制网页的布局、颜色、字体等视觉表现。本章将系统学习 CSS 的核心概念和现代布局技术。

## 学习目标

- 掌握 CSS 选择器和盒模型基础
- 解现代布局方案（Flexbox 和 Grid）
- 学会创建流畅的动画和过渡效果
- 掌握响应式设计原则
- 能够编写模块化、可维护的 CSS 代码

## css 版本演进

- CSS1 (1996) - 基础样式控制
- CSS2 (1998) - 定位和媒体类型
- CSS2.1 (2004-2011) - 广泛支持的稳定版本
- CSS3 (模块化发展) - 现代 CSS 标准
- CSS4 (草案阶段) - 选择器和功能增强

## 核心概念

CSS 工作基于以下基本机制：

- 层叠（Cascade）
- 解决样式冲突的规则
- 继承（Inheritance）
- 子元素继承父元素样式
- 盒模型（Box Model）
- 内容、内边距、边框、外边距
- 视觉格式化模型（Visual Formatting Model）
- 元素如何排列和显示

## 学习建议

- 渐进学习：从基础选择器开始，逐步掌握高级特性
- 实践验证：每个布局技术都要亲手构建示例
- 理解原理：掌握浏览器渲染机制，理解重绘和回流
- 移动优先：从移动端布局开始，逐步增强到大屏幕
- 性能意识：学习优化 CSS 性能的最佳实践
